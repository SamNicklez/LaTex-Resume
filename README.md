### README for the LaTeX Resume Repository

#### Repository Overview
The [LaTeX Resume](https://github.com/SamNicklez/latex-resume) repository contains the LaTeX source code for a professional resume. This repository is a straightforward and effective tool for anyone looking to create a resume using LaTeX.

#### Project Description
This project comprises a single LaTeX file, `Resume.tex`, which serves as a template for a resume. The LaTeX code is structured to be easily modifiable, allowing users to personalize their resume with their own information and experiences.

#### Key Features
- **LaTeX Resume Template**: The [`Resume.tex`](https://github.com/SamNicklez/latex-resume/blob/main/Resume.tex) file provides a structured and professional template for a resume.
- **Customizable Sections**: The template includes sections for education, experience, technical skills, and leadership/extracurricular activities.
- **Professional Formatting**: The template employs a clean and professional layout, suitable for various professional fields.

#### Getting Started
1. **Editing the Resume**:
   - Download the [`Resume.tex`](https://github.com/SamNicklez/latex-resume/blob/main/Resume.tex) file.
   - Use a LaTeX editor to modify the file, adding personal details, educational background, work experience, skills, and other relevant information.

2. **Compiling the Resume**:
   - Compile the `.tex` file into a PDF using a LaTeX compiler.
   - Online LaTeX editors can also be used for editing and compiling.

#### Project Structure
- [`Resume.tex`](https://github.com/SamNicklez/latex-resume/blob/main/Resume.tex): The main LaTeX file containing the resume template.
